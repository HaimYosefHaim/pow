834 // A library's status is an 8-bit value with information about it.
835 // The high bit indicates whether the library is present (1), or if it is a
836 // stub (0). The lower 7-bits can be used for codes specific to the library.
837 // This allows a library, like the security library, to specify what additional
838 // features are present.
839 // A value of 0xFF is reserved, it indicates an error in retrieving the
840 // library status.
