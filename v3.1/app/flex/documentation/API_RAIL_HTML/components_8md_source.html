4 ## Software Components
5 
6 These are some of the RAIL software components available for application use.
7 
8 ### Libraries
9 RAIL library software components provide the ability to choose support for
10 single or multiple protocols. Current components include:
11 * \ref rail_lib "RAIL Library, Single Protocol"
12 * \ref rail_lib "RAIL Library, Multiprotocol"
13 
14 ### Utilities
15 Radio utility software components provide optional source code that aid with
16 radio setup and functionality. Current components include:
17 * \ref rail_util_aox "Angle of Arrival/Angle of Departure (AoX)"
18 * \ref rail_util_ant_div "Antenna Diversity"
19 * \ref rail_util_callbacks "Callbacks"
20 * \ref rail_util_coex "Coexistence"
21 * \ref rail_util_dma "Direct Memory Access (DMA)"
22 * \ref fem_util "Front End Module (FEM)"
23 * \ref rail_util_init "Initialization"
24 * \ref rail_util_pti "Packet Trace Interface (PTI)"
25 * \ref rail_util_pa "Power Amplifier (PA)"
26 * \ref rail_util_protocol "Protocol"
27 * \ref rail_util_recommended "Recommended"
28 * \ref rail_util_rf_path "RF Path"
29 * \ref rail_util_rssi "Received Signal Strength Indicator (RSSI)"
30 
31 ### RAILtest Application
32 RAILtest application software components are specific to the RAILtest
33 application. Current components include:
34 * \ref rail_test_core
35 * \ref rail_test_graphics
