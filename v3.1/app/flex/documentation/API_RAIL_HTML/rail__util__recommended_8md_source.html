4 ## RAIL Recommended Utility
5 This optional software component can be enabled to include a recommended
6 collection of other optional RAIL utilities.
7 
8 This RAIL utility attempts to identify those RAIL utilities most commonly used
9 by applications. If a list of utilities different from what is auto-selected
10 here is sought, this component must be removed from the project with the
11 custom list manually being added to the project.
12 
13 Common RAIL utilities recommended for general use include:
14 * \ref rail_util_init "Initialization"
15 * \ref rail_util_callbacks "Callbacks"
16 * \ref rail_util_protocol "Protocol"
17 * \ref rail_util_pa "Power Amplifier (PA)"
18 * \ref rail_util_pti "Packet Trace Interface (PTI)"
19 * \ref rail_util_rssi "Received Signal Strength Indicator (RSSI)"
20 * \ref rail_util_dma "Direct Memory Access (DMA)"
21  * This primarily accelerates radio initialization, so it is currently
22  included only for dynamic multiprotocol (DMP) scenarios.
