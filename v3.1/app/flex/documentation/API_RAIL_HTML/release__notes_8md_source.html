36 ## RAIL Library 2.10.0 GA
37 
38 December 9, 2020
39 
40 ### Release Highlights
41 
42 * Added a new \ref rail_util_rssi component that can be used to set the correct
43  die specific RSSI offset for your chip. If enabled this should shift the RSSI
44  values on the EFR32xG1, EFR32xG12, EFR32xG13, EFR32xG14, and EFR32xG21 devices
45  to make them more accurate. If you have previously compensated for this be
46  sure to stop doing that before using this new component.
47 * Added a new \ref RAIL_SetAddressFilterAddressMask() API to allow the user to
48  mask out certain bits when doing address filtering and not match the whole
49  address.
50 * Added support for MGM210PB22JIA, MGM210PB32JIA, BGM210PB22JIA and
51  BGM210PA32JIA modules
52 * Added support for the Z-Wave Long Range PHYs and packet formats.
53 * Added support for the Quuppa PHYs for location tracking.
54 * Bugfixes and minor improvements
