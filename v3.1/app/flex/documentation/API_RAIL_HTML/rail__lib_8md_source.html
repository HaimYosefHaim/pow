4 ## RAIL Multiprotocol Library
5 The RAIL multiprotocol library allows multiple protocols to be initialized,
6 set up, and managed at the library level. Dynamic Multiprotocol (DMP) requires
7 the use of the RAIL multiprotocol library. Addition information can be found
8 here: \ref rail_multiprotocol.
9 
10 ## RAIL Single Protocol Library
11 The RAIL single protocol library reduces flash and memory footprints compared
12 to the RAIL multiprotocol library. The \ref RAIL_SchedulerInfo_t input
13 parameter, used by select RAIL library API functions, is always ignored in
14 the RAIL single protocol library. The single protocol library can be used
15 for Switched Multiprotocol (SMP) because the RAIL library is
16 initialized and used for one wireless protocol at a time.
